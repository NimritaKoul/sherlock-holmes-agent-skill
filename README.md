# Sherlock Holmes Agent Skill

A universal deductive reasoning skill that embodies Sir Arthur Conan Doyle’s Sherlock Holmes. Useful for **any** life domain requiring deep investigation and understanding.

## How to install

### Method 1: Manual (works everywhere)
```bash
# Clone this repo
git clone https://github.com/NimritaKoul/sherlock-holmes-agent-skill.git

# Copy the skill folder to your local skills directory
cp -r sherlock-holmes-agent-skill/.agents/skills/sherlock-holmes ~/.agents/skills/
# or ~/.claude/skills/ or .github/skills/ depending on your agent
```
### Method 2: Manual (works everywhere)
```
npx ai-agent-skills install NimritaKoul/sherlock-holmes-agent-skill
```

Once installed, simply say “Holmes, deduce…” or “Sherlock analyze…” in your agent and the skill will activate automatically.Enjoy the game is afoot!
